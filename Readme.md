# Income & Expense Tracker
A simple web application to track your income and expenses in real-time.

## Features
- Add income and expense transactions
- View total balance
- See transaction history
- Responsive design for desktop and mobile

## Technologies
- React
- Bootstrap
- JavaScript
- MongoDB
- Node.js
- Express.js
